# Dropdown Component

Pull request: https://github.com/rxdi/ui-kit/pull/20

##### Usage

```typescript
import { DropdownComponent } from '@rxdi/ui-kit/dropdown';

@Module({
  components: [DropdownComponent]
})
export class AppModule {}
```
